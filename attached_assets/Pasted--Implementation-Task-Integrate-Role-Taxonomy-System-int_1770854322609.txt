## STEP 3: EDIT `backend/enhancedEnrichmentService.ts`

This is the orchestrator that runs the Apollo search pipeline. We need to call the taxonomy lookup BEFORE the AI-powered department inference.

### What to do:

1. Add this import at the top of the file:
```typescript
import { classifyJobRole, getRecruiterTitleVariations } from './roleTaxonomyService';
```

2. Find the section in the main orchestration function where the job title has been extracted (after `extractJobData()` or `extractApolloSearchParams()` has run) and BEFORE the department inference / Apollo search begins.

3. Add the taxonomy fast-lookup at that point:
```typescript
// --- TAXONOMY FAST-LOOKUP (runs before AI inference) ---
const taxonomyResult = classifyJobRole(jobTitle, jobDescription);

let departmentFromTaxonomy: string | null = null;
let hiringManagerTitlesFromTaxonomy: string[] | null = null;
let apolloDepartmentsFromTaxonomy: string[] | null = null;
let seniorityFromTaxonomy: string | null = null;

if (!taxonomyResult.shouldUseAI && taxonomyResult.taxonomyMatch) {
  // High-confidence taxonomy match — use directly
  departmentFromTaxonomy = taxonomyResult.taxonomyMatch.department;
  hiringManagerTitlesFromTaxonomy = taxonomyResult.taxonomyMatch.hiringManagerTitles;
  apolloDepartmentsFromTaxonomy = taxonomyResult.taxonomyMatch.apolloSearchDepartments;
  seniorityFromTaxonomy = taxonomyResult.taxonomyMatch.apolloSeniority;
  
  console.log(`[Taxonomy] High-confidence match: ${taxonomyResult.taxonomyMatch.departmentLabel} (${taxonomyResult.taxonomyMatch.confidence.toFixed(2)}) — skipping AI department inference`);
}
// If shouldUseAI is true, we proceed with existing AI inference below
// but pass taxonomyResult.taxonomyContext into the prompt for better accuracy
```
